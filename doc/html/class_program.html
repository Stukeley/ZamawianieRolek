Statyczne metody publiczne

static void Main ()
 Punkt wejściowy aplikacji. Funkcja dokonuje serializacji oraz deserializacji bazy danych, oraz wyświetla użytkownikowi menu aplikacji.
 

Właściwości

static bool _isLoggedIn [get]
 Właściwość zwracająca wartość true, jeżeli użytkownik jest zalogowany.
 
static bool _isUserProfileSelected [get]
 Właściwość zwracająca wartość true, jeżeli użytkownik wybrał profil.
 

Statyczne metody prywatne

static void RegisterNewUser ()
 Funkcja wywoływana w przypadku próby rejestracji użytkownika. Wyświetla na ekranie konsoli stosowne informacje oraz pobiera od użytkownika niezbędne do rejestracji dane.
 
static void LoginUser ()
 Funkcja wywoływana w przypadku próby zalogowania użytkownika. Wyświetla na ekranie konsoli stosowne informacje oraz pobiera od użytkownika niezbędne do logowania dane.
 
static void CreateUserProfile ()
 Funkcja wywoływana w przypadku próby stworzenia nowego profilu użytkownika. Wyświetla na ekranie konsoli stosowne informacje oraz pobiera od użytkownika niezbędne do utworzenia profilu dane.
 
static void SelectUserProfile ()
 Funkcja wywoływana w przypadku próby wyboru profilu użytkownika. Wyświetla na ekranie konsoli stosowne informacje oraz pobiera od użytkownika niezbędne dane.
 
static void DisplayAllSheds ()
 Funkcja wywoływana w przypadku próby wyświetlenia na ekranie listy dostępnych wiat.
 
static void LendSkates ()
 Funkcja wywoływana w przypadku próby rozpoczęcia wypożyczenia wrotek przez użytkownika. Wyświetla na ekranie konsoli stosowne informacje oraz pobiera od użytkownika niezbędne do wypożyczenia dane.
 
static void FinishRide ()
 Funkcja wywoływana w przypadku próby oddania przez użytkownika wypożyczonych wrotek. Wyświetla na ekranie konsoli stosowne informacje, w tym koszt przejazdu.
 
static void SwitchTurbo ()
 Funkcja wywoływana w przypadku próby przełączenia trybu turbo w wypożyczonych wrotkach. Jeżeli tryb turbo jest włączony, zostanie on wyłączony; jeżeli nie, wówczas zostanie on włączony.
